## 安装

让你的 Agent（Claude Code / Codex / OpenClaw 等）执行：

```
帮我安装这个 skill：https://github.com/lushouhei/lushouhei-skills/tree/main/<skill-name>
```

把 `<skill-name>` 换成具体的 skill 目录名即可。Agent 会自己 clone 到对应目录，不用你操心路径。
